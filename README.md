## 3.現場の登録
検索ボタンをクリックすると近いところにマップのピンが飛びます

<img width="1052" alt="image" src="https://github.com/KantaHasegawa/mizuho-manual/assets/66783124/86b93d43-dc6f-4a73-a0bb-4730fb5797f6">
